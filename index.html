Welcome

My name is Joseph Gibli and I go to Carnegie Mellon University. I'm majoring in Statistics and Machine Learning, and on track to double in Computer Science.

Feel free to check out some of my projects, data visualizations, and short stories!

Projects

GenAMap

A genomics data visualization web app making machine learning accessible to biology researchers. I am working on front-end development designing interactive d3.js visualizations for genetic data.

This has been a unique challenge because browsers are not equipped to handle visualize large data sets using standard approaches. Also, I am learning how to develop in a fast-paced environment.

Habeas Corpus

Research project under the Machine Learning Department at Carnegie Mellon. I developed an ontology builder that used a modified Naive Bayes algorithm in MATLAB to predict the format of an ontology from labeled examples. The user can modify the ontology in order to speed up the research process.

This project has taught me how to apply machine learning theory and build on ideas learned in the classroom. I also learned how to assess and present findings to a team.

ShellLab

Course project for Introduction to Computer Systems (15-213). I wrote a terminal shell entirely in C that managed job processes, signals, and file input/output.

This was one of my favorite labs in the course because I developed something that I could interact with, but understand at the low level.

CacheLab

Another course project for Introduction to Computer Systems (15-213). I wrote an LRU cache simulator in C that would take in a trace file and dynamically allocate and free cache memory.

I thought this was such an interesting lab! I learned at a deeper level how to strategically access memory for efficency and organize my code for readability.
