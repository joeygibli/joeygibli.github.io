Projects

GenAMap

A genomics data visualization web app making machine learning accessible to biology researchers. I am working on front-end development designing interactive d3.js visualizations for genetic data.

This has been a unique challenge because browsers are not equipped to handle visualize large data sets using standard approaches. Also, I am learning how to develop in a fast-paced environment.

Habeas Corpus

Research project under the Machine Learning Department at Carnegie Mellon. I developed an ontology builder using d3.js and MATLAB to predict the format of an ontology from labeled examples. The user can modify the ontology in order to speed up the research process.

This project has taught me how to apply machine learning theory and build on ideas learned in the classroom. I also learned how to assess and present findings to a team.

Data Visualizations

Dynamic Clustering

(Still in development) An interactive visualization that allows users to explore how data points cluster according to different criterion. This example data set are individuals shot and killed by police in 2016 (source: Washington Post).

I wanted to develop an unbiased, interactive, high dimensional look at individuals killed by police. Of course, circles on a screen cannot possibly tell the whole story. This tool reflects numbers, not opinions.
